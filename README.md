# html2pdf
HTML to PDF converter provided by command line interface and RESTful API. Powered by `puppeteer`.

## Installing
```
npm install
```

## Usage

### Command-line
```
npm link
html2pdf "https://google.com" -o google.pdf
```

### RESTful API
```
npm start
```
